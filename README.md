## ðŸ“± User Flows

### For Tree Planters

1. **Register**: Create account and link OneWallet
2. **Plant**: Plant trees following best practices
3. **Submit**: Capture GPS coordinates + photo evidence via app

4. **Report**: Receive verified carbon offset certificates
5. **Brand**: Showcase environmental impact in ESG reports

## Common Issues & Troubleshooting

### Issue: "Module not found" errors

**Solution**: Delete `node_modules` and reinstall:

```bash
rmdir /s /q node_modules
del package-lock.json
npm install
```

### Issue: Port 3000 already in use

**Solution**: Kill the process using port 3000 or use a different port:

```bash
# Use a different port
set PORT=3001 && npm start
```

### Issue: Hardhat compilation fails

**Solution**: Clear Hardhat cache and recompile:

```bash
npx hardhat clean
npx hardhat compile
```

### Issue: Rust build fails

**Solution**: Update Rust toolchain:

```bash
rustup update
```
